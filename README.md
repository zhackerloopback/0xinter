# OSINT Threat Intelligence Platform

Enterprise-grade OSINT and Threat Intelligence Platform built with Flask.

## Features

- WHOIS Lookup
- DNS Lookup
- GeoIP Lookup
- Hash Analysis
- Metadata Analysis
- VirusTotal Integration
- AlienVault OTX
- Shodan Integration
- Threat Hunting
- IOC Correlation
- Case Management
- Real-Time Alerts

## Installation

pip install -r requirements.txt

python app.py
